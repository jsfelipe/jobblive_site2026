# Copywriting Rules

## Objective

All copy must be designed to:

- Increase clarity
- Improve conversion
- Strengthen positioning
- Build trust
- Guide user attention
- Support SEO and GEO
- Improve readability
- Create emotional connection without exaggeration

The writing should feel:

- Premium
- Strategic
- Human
- Modern
- Confident
- Clear

---

# Core Principles

Always prioritize:

1. Clarity
2. Simplicity
3. Specificity
4. Readability
5. User intent
6. Emotional relevance
7. Conversion

---

# Tone of Voice

The tone should feel:

- Premium
- Minimal
- Intelligent
- Direct
- Modern
- Confident
- Calm

---

# Avoid Tone That Feels

- Generic
- Exaggerated
- Aggressive
- Overhyped
- Robotic
- Corporate
- Artificial

---

# Writing Style Rules

## Prefer:

- Short sentences
- Clear structure
- Strong verbs
- Concrete language
- Natural rhythm

## Avoid:

- Long paragraphs
- Buzzwords
- Empty claims
- Repetition
- Complex wording

---

# Clarity Rules

Every sentence should answer at least one:

- What is this?
- Why does it matter?
- What problem does it solve?
- What action should the user take?

---

# Readability Rules

## Use:

- Short paragraphs
- Clear spacing
- Section hierarchy
- Lists when useful
