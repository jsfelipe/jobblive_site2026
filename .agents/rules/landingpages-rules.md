# landing-page-rules.md


## Objective

Landing pages must maximize:

- Conversion
- Clarity
- Trust
- Speed
- Readability
- User attention

Every section must have a purpose.

---

# Core Principles

Landing pages should feel:

- Premium
- Direct
- Trustworthy
- Modern
- Focused

---

# Structure Rules

Recommended order:

1. Hero
2. Social Proof
3. Benefits
4. Services / Features
5. Problem
6. Solution
7. Results
8. Testimonials
9. FAQ
10. CTA

---

# Hero Section Rules

The hero must communicate:

1. What you do
2. Who it is for
3. Main benefit
4. Primary CTA

All visible above the fold.

---

# Headline Rules

## Headlines must:

- Be direct
- Focus on transformation
- Be easy to scan
- Avoid generic buzzwords
